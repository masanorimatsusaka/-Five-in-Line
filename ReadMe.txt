/********************************************/
「五目並べ」
/********************************************/
・概要
Assetの「PUN2」を用いたオンライン対戦用の五目並べが遊べます。
一つのパソコンで試される場合は
exeファイルを二回実行して二窓で行ってください。

・遊び方の流れ
１、exeファイルを実行してください。
２、ログイン画面が表示されます。適当な名前をクリックしてログインしてください。
３、ゲーム画面が表示されてから、右下にボタンが表示されたら
　　そのプレイヤーは「ルームリーダー」です。二人ログインしたらボタンが実行できるようになります。
　　右下にボタンが表示されない場合は「ルームメンバー」です。ルームリーダーがゲーム開始するまで待ってください。
４、ルームリーダーがゲームを開始したら試合が始まります。

ターンプレイヤーは右下の碁鉢から黒（または白）石をD&Dで碁盤においてください。
選択している碁石がわからなくなる、または碁石が碁盤外に落ちた場合は
やり直しボタンをクリックすることで、碁鉢の上に碁石が戻るようになります。

碁石の置き場が決定し、次へをクリックすれば
白（または黒）のプレイヤーに切り替わります。

また右上には今現在で３，４の列が存在するかどうかが出力されます。

＊特徴
碁石を十字の真ん中に置かなかった場合、ターン終了時に補正をかけて
十字の真ん中に置くよう修正しております。
